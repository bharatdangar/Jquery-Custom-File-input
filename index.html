Jquery Custom File Input Plugin

How to use

Add scripts in to Head 

<script src="js/jquery.js" type="text/javascript"></script> 
<script src="js/customfileinput.js" type="text/javascript"></script>

Add scripts in to Head 

<script type="text/javascript">
   $(document).ready( function(){
	  $('.customfileinput').customFileinput({
	  	  buttiontext : 'Browse',
		  customboxwidth : 234
	  });
   });
</script>

  • Mozila Firefox 3 +
  • Apple Safari 3 +
  • Google Chrome 5 +
  • Opera 8 +
